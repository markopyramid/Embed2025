Chat Search Feature

Pyramid 2025 introduces an enhanced Chat Search feature, integrated directly into the application interface. This allows users to interact using natural language to:

  • Quickly find existing reports, dashboards, and data models.
  • Ask questions directly about their data (e.g., "What were the total sales for last quarter?").
  • Refine searches and discover relevant content more intuitively.

This powerful NLQ capability streamlines data discovery and analysis within the Pyramid platform.

Learn more about Chat Search in the official documentation.
